# Daily Reports

> Status: Current
> Last reviewed: 2026-07-09
> Source of truth: Yes

This folder stores historical end-of-session reports for RiskGuard Trader.

Daily reports are not current product requirements. Use them to understand what changed, what was tested, what risks remain, and what should happen next.

## Naming Convention

Use:

```text
YYYY-MM-DD-short-description.md
```

## Report Sections

```md
# Daily Report - YYYY-MM-DD

> Status: Historical
> Last reviewed: YYYY-MM-DD
> Source of truth: No

## Session Summary
## Actions Taken
## Files Changed
## Issues Faced
## Tests And Checks
## Decisions Made
## Current Git State
## Risks Or Open Questions
## Recommended Next Steps
```
